- **[Services](lumify.api.Services.yml)** – application services
- **[Logic](lumify.api.Logic.yml)** – business logic
- **[Interfaces](lumify.api.Interfaces.yml)** – abstractions / contracts
- **[Models](lumify.api.Models.EF.yml)** – data models (EF Core entities & DTOs)
